## Project

Personal portfolio website. Currently in initial setup — no framework or build system has been chosen yet.

## Tech Stack

- **Framework**: Astro 4.x (SSG)
- **Styling**: Tailwind CSS 3.x
- **Language**: TypeScript (strict)
- **Hosting**: GitHub Pages

## Getting Started

```bash
# Install dependencies
npm ci

# Start dev server (http://localhost:4321)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Type check
npm run check

# Lint
npm run lint
```
